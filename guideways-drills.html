A · Slideway reactions & face width

Free-body of the carriage → three face reactions → size the width on the largest one. Reactions: B = Py; C = (Pzyp − Pyh)/b + G/2; A = Pz + G/2 − (Pzyp − Pyh)/b. Width c′ = A/([pav]·L). Pull Q = Px + f(Pz+Py+G).

A1 Slideway width MEHTA 4.2

Turning, Pz=300, Py=130, Px=120 kgf; carriage G=150 kgf, length L=30 cm. Two flat CI ways. Find the face width. (Assume way spacing b, tool height h, force position yp, f, [pav].)

Show solution ▸
What goes where Pz =300, Py =130, Px =120, G =150 · assume b =30, h =20, yp =15, f =0.1, [pav] =12.5 (=½·25 CI), L =30

moment term m = (Pzyp − Pyh)/b:

m = (300·15 − 130·20)/30 = (4500−2600)/30 = 63.3

A = Pz + G/2 − m (largest reaction):

A = 300 + 75 − 63.3 = 311.7 kgf

c′ = A/([pav]·L) · Q = Px + f(Pz+Py+G):

c′ = 311.7/(12.5·30) = 0.83 cm  |  Q = 120 + 0.1·580 = 178 kgf

width ≈ 0.83 cm (round to 1 cm); Q ≈ 178 kgf. All reactions +ve → no lift-off.

B · Saddle lift-off (V + flat ways)

Saddle lifts off the V-face A when reaction A → 0. Condition: (d+b)/2b = (Py/Pz)·tanα = tanθ·tanα, where θ = angle of the resultant to the vertical, α = V half-angle. So d/b = 2·tanθ·tanα − 1.

B1 Lift-off ratio MEHTA 4.1

Parting cut (Px=0). μ=0.5, V half-angle α=50°, resultant cutting force at 40° to the vertical. At what d/b does the saddle lift off face A?

Show solution ▸
What goes where θ = 40° (resultant to vertical) → tanθ = 0.839
α = 50° (V half-angle) → tanα = 1.192
μ does NOT enter (friction acts tangentially, not on the normal reaction)

(d+b)/2b = tanθ·tanα:

= 0.839 × 1.192 = 1.00 → d/b = 2(1.00) − 1 = 1.0

d/b = 1.0 — saddle lifts when the V-offset equals the way spacing. (Including the Py·h/b moment with h/b=0.5 lowers it to ≈0.16.)

B2 Lift-off ratio VARIANT of 4.1

Resultant at 35° to the vertical, V half-angle α=45°. Find the lift-off d/b.

Show solution ▸
What goes where θ = 35° → tanθ = 0.700 · α = 45° → tanα = 1.000

(d+b)/2b = 0.700 × 1.000 = 0.700 → d/b = 2(0.700) − 1 = 0.40

d/b = 0.40.

C · Pulling force — conventional vs ball ways

Conventional (sliding): Q = Px + f(Pz+Py+W). Ball (rolling): Q = Px + n·Fc + (fr/req)·N, with N = Pz+W, fr=0.001 cm, req=0.357, n=4, Fc=0.5. The saving is all in the friction term; Px is irreducible.

C1 Conventional vs ball MEHTA 4.8

Pz=300, Py=130, Px=120 kgf; carriage W=150 kgf; sliding f=0.15; ball dia 10 mm. Find the pull for each way.

Show solution ▸
What goes where Px =120 (in both), sum(Pz+Py+W) =580 · f =0.15
ball: N=Pz+W =450, n =4, Fc =0.5, fr/req =0.001/0.357

Conv: Q = 120 + 0.15·580 = 207 kgf

Ball: F = 4·0.5 + (0.001/0.357)·450 = 2.0 + 1.26 = 3.26 → Q = 120 + 3.26 = 123 kgf

207 vs 123 kgf → ~40% lower.

Ball: F = 2.0 + (0.001/0.357)·600 = 2.0 + 1.68 = 3.68 → Q = 160 + 3.68 = 163.7 kgf

D · Lead screw — pressure, shear stress, pitch error

Order: Q = Px+f(Pz+Py+G) → pav = 2Q/(πdL) (d=effective, L=nut) → torque Mt=(Qd/2)tan(λ+φ′), λ=tan⁻¹(t/πd), φ′=tan⁻¹(μ/cosα), α=14.5° → σ=4Q/(πdm²), τ=16Mt/(πdm³) on minor dm=OD−t → τmax=½√(σ²+4τ²) → pitch error Δt=Qt/(EAm).

D1 Lead screw MEHTA 4.9

Acme screw: OD=70, effective d=65, pitch t=10, nut L=100 (mm); Pz=150, Py=50, Px=45 kgf, μ=0.2, G=50 kgf, E=2.1×10⁶. Find pav, τmax, pitch error.

Show solution ▸
What goes where Q: Px =45, μ =0.2, sum(Pz+Py+G) =250 · pav: d =6.5 cm, L =10 cm
stresses on minor dm =OD−t=7−1=6.0 cm (not the effective dia!) · Am =πdm²/4=28.27, t =1 cm

Q = 45 + 0.2·250 = 95 kgf

pav = 2·95/(π·6.5·10) = 0.93 kgf/cm²

Mt ≈ 79.7 kgf·cm → σ=4·95/(π·6²)=3.36, τ=16·79.7/(π·6³)=1.88 → τmax=½√(3.36²+4·1.88²) = 2.52 kgf/cm²

Δt = 95·1/(2.1×10⁶·28.27) = 1.6×10⁻⁶ cm (0.016 µm)

pav=0.93, τmax=2.52 kgf/cm², Δt≈0.016 µm — all far within limits.

E · Hydrostatic pad — pressure & capacity

Always start: pocket pressure p₀ = P/(A·CL), A = pad area, CL = load coefficient (≈0.3–0.55). Capillary must drop pp − p₀. Capacity (orifice/upper bound) P = p₀·A·CL with p₀ → pp. Max stiffness at p₀/pp = 2/3.

E1 Pocket pressure MEHTA 4.6

Pad 50×80 mm, pocket 10×60 mm, load P=50 kgf, pump pp=40 kgf/cm², capillary l/d=20. Find pocket pressure (CL=0.542).

Show solution ▸
What goes where A =5×8=40 cm², CL =0.542 · P =50 (on top) · pp =40

p₀ = P/(A·CL) = 50/(40·0.542) = 50/21.7 = 2.31 kgf/cm²

restrictor drop = pp − p₀ = 40 − 2.31 = 37.7 kgf/cm²; size via crc with lc=20·dc

p₀ = 2.31 kgf/cm², CL = 0.542 — the graded quantities.

E2 Load capacity (orifice) MEHTA 4.7

Same pad (A=40 cm², CL=0.542) fed by an orifice, pump pp=80 kgf/cm², minimum film 0.01 mm. Find the load capacity.

Show solution ▸
What goes where A =40 cm², CL =0.542 · small film → pocket highly restrictive → p₀ → pp =80

P = p₀·A·CL, with p₀≈pp=80 → P = 80·40·0.542 = 1734 kgf (upper bound)

Capacity ≈ 1734 kgf — the 0.01 mm film makes p₀ sit close to pp.

E3 Pocket pressure VARIANT of 4.6

Pad 60×100 mm, pocket 15×70 mm, load P=80 kgf, pump pp=50 kgf/cm². Assume CL=0.50. Find p₀.

Show solution ▸
What goes where A =6×10=60 cm², CL =0.50 · P =80 · pp =50

p₀ = 80/(60·0.50) = 80/30 = 2.67 kgf/cm²

restrictor drop = 50 − 2.67 = 47.3 kgf/cm²

p₀ = 2.67 kgf/cm². (Check p₀/pp=0.053 ≪ 2/3 → low pump utilisation, as in real low-load pads.)

F · Composite (hydrodynamic) slider bearing — length

Two steps: (1) total bearing area from the pressure limit A = W/[pmax]; (2) width b₀ from the lifting equation Q = 0.133·μ·v·b₀³/hmin² (convert ν·ρ → μ, ÷9.81 for kgf·s/m²). Then total length = A/b₀, and each of n sliders = total length ÷ n.

F1 Slider length MEHTA 4.4

Load W=700 kgf, sliding v=2 m/min, oil ν=20 cSt, ρ=0.9 g/cm³, [pmax]=2 kgf/cm², minimum film hmin=0.02 mm, six composite sliders of equal width. Find width & length of each.

Show solution ▸
What goes where W =700, [pmax] =2 → area · per slider load =700/6=116.7 · μ=ν·ρ =1.835×10⁻³ kgf·s/m², v =0.0333 m/s, hmin =2×10⁻⁵ m

A = W/[pmax] = 700/2 = 350 cm²

b₀³ = Q·hmin²/(0.133·μ·v) → b₀ ≈ 17.9 cm (width)

total length = A/b₀ = 350/17.9 = 19.6 cm → each = 19.6/6 ≈ 3.3 cm

width ≈ 17.9 cm; each of 6 sliders ≈ 3.3 cm long. (Pressure limit sets the area; the hydrodynamic equation sets the width.)

F2 Slider length VARIANT of 4.4

Load W=900 kgf, [pmax]=2.5 kgf/cm², eight equal sliders, same oil (ν=20 cSt, ρ=0.9), v=2.5 m/min, hmin=0.02 mm.

Show solution ▸
What goes where W =900, [pmax] =2.5 → area · per slider =900/8=112.5 · μ same, v =0.0417 m/s

A = W/[pmax] = 900/2.5 = 360 cm²

b₀³ = Q·hmin²/(0.133·μ·v) with Q=112.5 → b₀ ≈ 16.4 cm

total length = 360/16.4 = 22.0 cm → each = 22.0/8 ≈ 2.7 cm

area 360 cm²; width ≈ 16.4 cm; each of 8 sliders ≈ 2.7 cm. (Same two-step method; width from the lifting equation.)

G · Recirculating ball screw — load capacity

Chain of three: per ball P = 200·db² (db in cm) → static pull Q = 100·Zr·db², Zr=0.7·Zt (only 70% loaded) → dynamic Qdyn=Q/k, k=0.9·(60·T·n·f/10⁷)^⅓, f=Zt/2. Watch the cube root and Zr.

G1 Ball-screw load MEHTA 4.10

Ball dia db=10 mm, total balls Zt=120, screw n=40 rpm, service life T=10000 h. Find load per ball and the static & dynamic pull.

Show solution ▸
What goes where db =1 cm (mm→cm!) · Zt =120 → Zr=0.7·120 =84 · f=Zt/2 =60, n =40, T =10000

per ball P = 200·1² = 200 kgf

static Q = 100·84·1² = 8400 kgf

k = 0.9·(60·10000·40·60/10⁷)^⅓ = 0.9·(144)^⅓ = 4.72 → Qdyn = 8400/4.72 = 1781 kgf

per ball 200, static 8400, dynamic ≈1781 kgf.
