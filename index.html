CLO1 — Drives, Working Motions, Kinematics

Identify drives, the kinematic chain, and the working motions of machine tools. Numerical core = cutting speed.

Working motions = (1) primary / cutting motion (provides the cutting speed) + (2) feed motion (advances the tool/work to remove the next layer). Auxiliary motions = clamping, indexing, rapid traverse — no chip.

Drive types

  • Individual (separate) drive — one motor for one machine tool → independent operation, the modern standard. (Part-A blank #2 answer.)
  • Group / line-shaft drive — one motor drives several machines via a common shaft & belts (obsolete).
  • Multi-motor drive — separate motors for spindle, feed, rapid traverse, coolant on one machine.

Worked — Cutting speed ↔ spindle rpm CLO1

Given: Turn a workpiece of diameter D = 100 mm. Recommended cutting speed V = 40 m/min.

Find: required spindle speed N; and if the machine's nearest available speed is 125 rpm, the actual cutting speed.

Show solution ▸

Formula. V = πDN/1000 (D in mm, N in rpm, V in m/min) → N = 1000V/(πD).

Substitute. N = 1000×40/(π×100) = 40000/314.16 = 127.3 rpm.

Actual speed at N = 125 rpm. V = π×100×125/1000 = 39.3 m/min.

Answer: N ≈ 127 rpm → pick 125 rpm → actual V ≈ 39.3 m/min.

Exam tip: V=πDN/1000 is the single most-used CLO1 numerical. Available speeds come from the GP gearbox (see CLO2) — that's why the geometric step matters.

Kinematic chain (Q7a style answer). In a lathe turning centre: speed (main) chain = motor → speed gearbox → spindle → workpiece (sets cutting speed). Feed chain = spindle → feed gearbox / lead-screw or feed-rod → carriage/cross-slide (sets feed, synchronised to spindle for threading). Each "chain" is the train of gears/screws transmitting one working motion.

CLO2 — Regulation of Speed & Feed (Stepped Drives)

Why geometric progression (GP)? Spindle speeds in a stepped gearbox are arranged in a geometric progression so the percentage speed change between steps is constant. This keeps the maximum loss of cutting speed (and hence productivity loss) the same across the whole range — the optimal arrangement. (Part-A blank #3 = "geometric".)

Core formulas

common ratio φ = (n_max / n_min)^(1/(z−1)) speed steps n₁=n_min , n₂=n₁φ , n₃=n₁φ² , … , n_z = n_min·φ^(z−1) standard φ ∈ {1.06, 1.12, 1.26, 1.41, 1.58, 1.78, 2.0}

Prob (= Last Exam Q5) — 9-speed GP gearbox ★ ON LAST EXAM (Q5) CLO2

Given: Spindle gearbox for a medium-duty lathe. z = 9 speeds, nmin = 100 rpm, nmax = 1600 rpm, geometric progression, structural formula 9 = 3 × 3.

Find: common ratio φ; list the 9 speeds; explain the structural, ray and speed diagrams.

Show solution ▸

Step 1 — common ratio. φ = (nmax/nmin)^(1/(z−1)) = (1600/100)^(1/8) = 16^(1/8) = 1.41 (≈ √2; a standard step value).

Step 2 — the 9 speeds (multiply by 1.41 each step), rounded to the standard R20 series:

100 → 141 → 200 → 283 → 400 → 566 → 800 → 1131 → 1600 rpm
≈ standard: 100, 140, 200, 280, 400, 560, 800, 1120, 1600 rpm.

Step 3 — structural formula 9 = 3(1) × 3(3). Two gear stages in series: stage I has 3 ratios with characteristic x₁ = 1; stage II has 3 ratios with characteristic x₂ = 3. Total speeds = 3 × 3 = 9. The characteristic = number of speeds a stage "spans" before the next stage shifts.

Step 4 — the three diagrams.
• Structural (kinematic skeleton) diagram: shows the shafts and the number of gear pairs (transmissions) per stage — here 3 + 3 — i.e. the layout of the gearbox.
• Ray diagram (structural-net): each speed plotted on a log scale; rays connect input to output shafts, the slope of a ray = a transmission ratio. It checks that ratios stay within limits (1/4 ≤ i ≤ 2) and that steps are evenly spaced by φ.
• Speed diagram: the final spindle-speed chart — the 9 output speeds spaced by equal log-intervals (= GP), confirming uniform % change.

Answer: φ = 1.41; speeds 100,140,200,280,400,560,800,1120,1600 rpm; structural formula 9 = 3(1)×3(3); the structural diagram fixes the layout, the ray diagram fixes/validates the ratios, the speed diagram shows the equally-log-spaced output speeds.

Worked — number of teeth for a gear pair in the gearbox CLO2

Given: A gear pair must give ratio i = 1/φ = 1/1.41 = 0.71 (one downward step). The two meshing gears on adjacent shafts have a fixed centre distance → sum of teeth Z₁+Z₂ = constant = 84 (typical).

Find: the tooth numbers Z₁ (driver) and Z₂ (driven).

Show solution ▸

Step 1. i = Z₁/Z₂ = 0.71 and Z₁+Z₂ = 84.

Step 2. Z₁ = 0.71·Z₂ → 0.71Z₂ + Z₂ = 84 → 1.71Z₂ = 84 → Z₂ = 49.1 ≈ 49; Z₁ = 84−49 = 35.

Check. 35/49 = 0.714 ≈ 1/φ ✓ (keep within 1/4 ≤ i ≤ 2).

Answer: Z₁ = 35, Z₂ = 49 (sum 84, ratio ≈ 1/1.41).

Exam tip: all gears sharing a shaft pair must keep the same module, so Z₁+Z₂ = const fixes the centre distance. Solve the 2-equation system; round to whole teeth and re-check the ratio limit.

Chapter 3 — Design of Machine Tool Structures (CLO4)

Beds, columns, bases/tables. Core: torsion of open/closed thin sections, bending rigidity, stiffeners, stiffness. Prob 3.12 = exam Q6.

The two torsion section facts that win marks: Open thin section It = β·⅓Σbt³, τmax = M·tmax/It, θ = M·L/(G·It). Closed thin section τ = M/(2A₀δ), Mlim = 2τA₀δ. βI-section = 1.20, βchannel = 1.12.

Prob 3.1 CLO4

Given: Solid rectangular structure, height h, width b, length l, simply supported, loaded as Fig 3.32 (central load P + symmetric UDLs q₁, q₂). Permissible stress [σ], permissible deflection [δ], modulus E.

Find: Derive the optimum ratio l²/h.

Show solution ▸

Step 1 — strength volume. Mmax = Pl/4, zmax = h/2, I = bh³/12. σmax = (Pl/4)(h/2)/(bh³/12) = (3/2)·Pl/(bh²). Set σmax = [σ], metal volume Vσ = b·h·l: Vσ = (3/2)·(P/[σ])·(l²/h).

Step 2 — stiffness volume. δ = Pl³/(48EI). Set δ = [δ]: Vδ = (P/(4E[δ]))·(l²/h)².

Step 3 — optimum. Both criteria need the same metal: Vσ = Vδ → (3/2)(P/[σ])(l²/h) = (P/(4E[δ]))(l²/h)².

Step 4 — simplify. Cancel P and one (l²/h): (3/2)/[σ] = (1/(4E[δ]))·(l²/h).

Answer: (l²/h)opt = 6E[δ]/[σ]. If l²/h < opt → design for strength; if > opt (the usual machine-tool case) → design for stiffness.

Exam tip: derivation hinges on equating Vσ and Vδ; memorise (l²/h)opt = 6E[δ]/[σ].

Prob 3.2 ★ LIKELY CLO4

Given: Vertical milling machine. Carbide cutters: D = 50–250 mm, Z = 8–14. HSS: D = 80–250 mm, Z = 10–26. Cutting speed v = 30 m/min (HSS–MS) to 500 m/min (carbide–Al). Tooth excitation fe = n·Z/60, n = 1000v/(πD).

Find: excitation (tooth) frequency range to be avoided.

Show solution ▸

Lowest fe: largest D, lowest v, fewest Z. n = 1000×30/(π×250) = 38.2 rpm; fe,min = 38.2×8/60 ≈ 5.1 Hz.

Highest fe: smallest D, highest v, most Z. HSS D80,v500,Z26: n = 1989 rpm; fe = 1989×26/60 ≈ 862 Hz.

Separation rule: natural freq should be ≥ 2.5× the highest or ≤ lowest/2.5 → would need <2 Hz or >2155 Hz, both impractical.

Answer: forcing frequency sweeps ≈ 5 Hz → 862 Hz; this whole band must be avoided. Since design alone can't clear it, fit a tunable damper on the spindle/arbor.

Exam tip: use f = nZ/60 with n = 1000v/(πD) at both extremes; answer = the band 5–862 Hz + "add a damper".

Prob 3.3 CLO4

Given: Bed under torsion vs bed under bending. Compare closed box vs open (I/channel) section.

Find: prove why torsion beds are closed-box and bending beds can be open I-sections.

Show solution ▸

Torsion, closed: It = 4A₀²/(∮ds/δ), A₀ = enclosed area → huge It.

Torsion, open: It = β·⅓Σbt³ — no enclosed area, only thin strips cubed in small thickness → It,open ≪ It,closed (Table 3.3: box ~16× stiffer in torsion).

Bending: I = Σ(I + Ad²) — material far from neutral axis dominates; an open I places flanges at the extreme fibres just as well as a box (deflection nearly equal).

Answer: torsion → It ∝ A₀² so use closed box; bending → I ∝ Ad² (flange distance) independent of closure, so an open I-section does it with less metal.

Exam tip: one-line proof = torsion needs A₀² (closed), bending needs flange distance (open OK).

Prob 3.4 CLO4

Given: Channel welded to a box section (Fig 3.33), forming a compound open section; channel wall δ, added member walls 2δ. βchannel = 1.12. (Same method as worked Example 3.1.)

Find: factor by which torsional stiffness improves and shear stress changes.

Show solution ▸

Channel alone: It = 1.12·⅓·δ³·(2b + a).

Compound: It = 1.12·⅓·δ³·(3b + 8a) (added member thickness 2δ → (2δ)³ = 8δ³).

Stiffness factor: φ ratio = It,comp/It,chan = (3b + 8a)/(2b + a).

Shear stress: τ = M·δmax/It; channel δmax=δ, compound δmax=2δ → ratio = 2(2b+a)/(3b+8a) (<1 → stress drops).

Answer: torsional stiffness improves by (3b+8a)/(2b+a); shear stress changes by 2(2b+a)/(3b+8a). Plug the figure's b, a (=h) for numbers.

Exam tip: identical algebra to Example 3.1 — write It for both, stiffness = It ratio, stress carries the extra δmax=2δ.

Prob 3.5 CLO4

Given: Lathe bed L=200 cm, B=50 cm, wall height 100 cm; two vertical walls δ=5 cm; stiffeners 2.5 cm. CI, E=1.2×10⁶ kgf/cm².

Find: reduced bending rigidity EIbr for perpendicular vs diagonal stiffeners.

Prob 3.6 CLO4

Given: CI box column, outer 30×30 cm, wall 3 cm, h=200 cm; loads 100/100 kgf vertical + 200 kgf horizontal (Fig 3.34). E=1.2×10⁶ kgf/cm².

Find: maximum deflection; effect of a 10×30 cm aperture.

Show solution ▸

Section: I = (30⁴−24⁴)/12 = 39 852 cm⁴; A = 324 cm².

Aperture: a hole in the loaded (bending) wall cuts local I; Table 3.4 → stiffness ~85% → deflection rises ~15–18%.

Prob 3.12 ★ ON LAST EXAM (Q6) CLO4

Given: I-section (Fig 3.38): height 140 mm, flange 80 wide × 20 thick, web 10 thick. Mt = 1000 kgf·m = 100 000 kgf·cm. L = 100 cm. G = 8×10⁵ kgf/cm². βI = 1.20.

Chapter 4 — Design of Guideways & Power Screws (CLO3)

Functions/requirements of guideways, slideway sizing, hydrostatic/hydrodynamic bearings, lead & ball screws.

Guideway requirements (2-mark / Q7c answer): (1) high stiffness/rigidity, (2) low friction & good wear resistance, (3) accuracy of travel (geometric), (4) good damping, (5) easy clearance adjustment & protection from chips. Cutting force components: Pz tangential, Py radial, Px axial.

Prob 4.1 — Saddle lift-off from face A CLO3

Given: Parting on a lathe with V + flat slideways (Fig 4.10). μ = 0.5, V half-angle α = 50°, h/b = 0.5; resultant cutting force R at 40° to vertical; parting → Px = 0.

Find: d/b ratio at which the saddle just lifts off face A.

Show solution ▸

Resolve R: Pz = R cos40° = 0.766R; Py = R sin40° = 0.643R; Py/Pz = tan40° = 0.839.

Reaction A (Eq 4.11): lift-off when A → 0 → [Pz(d+b)/2b]cosα = Py sinα.

Rearrange: (d+b)/2b = (Py/Pz)·tanα = tan40°·tan50° = 0.839×1.192 = 1.000.

Solve: (d+b)/2b = 1 → d = b.

Answer: d/b = 1.0 (saddle lifts when V-offset = way spacing). Including the Py·h/b term lowers it to d/b ≈ 0.16. μ does not enter the lift-off condition.

Exam tip: clean form (d+b)/2b = tanθ·tanα; here the tangents are reciprocals → product 1 → d/b = 1.

Prob 4.3 — Best slideway combination (least radial deflection) CLO3

Given: flat–flat, flat–V, V–V combinations. Contact deflection δ = k·pav.

Find: which gives least radial deflection of the cutting edge? Prove it.

Show solution ▸

Model (Eq 4.36): δF = δB + (δA−δC)·h/b, with δ ∝ pav.

flat–flat: radial Py reacted by the narrow vertical face B (small area) → large δB → worst.

V-way: radial load shared on both inclined faces → effective width d = m cos²α + n cos²β → lower pressure; apex self-centres.

Answer: V–V least, flat–V intermediate, flat–flat worst. Replacing flat width by V width (cos²α factor) raises area, lowers pav, so δ ∝ pav drops in that order; V–V also self-aligns.

Exam tip: anchor on δ = k·pav + Eq 4.36, then argue area (V splits load over two faces). That earns the "proof" marks.

Prob 4.5 — Derivation: capillary-fed hydrostatic pad capacity CLO3

Given: rectangular pad B×L, pocket b×l, gap h, fed via capillary (pp), pocket pressure p₀.

Find: derive load capacity Pc.

Show solution ▸

Pad load: P = p₀·A·CL.

Pocket flow: Q = p₀h³/(μCF). Capillary flow: Q = (pp−p₀)/(μcrc).

Continuity (μ cancels): p₀h³/CF = (pp−p₀)/crc → p₀ = pp·CF/(CF + crch³).

Answer: Pc = pp·A·CL·CF/(CF + crch³). Stiffness max at p₀/pp = 2/3 → Kc,max = −Pc/h.

Exam tip: "equate pocket flow to restrictor flow, μ cancels, solve p₀, substitute back." Optimum p₀/pp = 2/3 is a classic follow-up.

Prob 4.6 — Capillary-compensated pad: viscosity & capillary size CLO3

Given: pad 50×80 mm² (B=5, L=8 cm), pocket 10×60 mm² (b=1, l=6 cm), load P=50 kgf, v=500 cm/min, h=0.01 mm, pp=40 kgf/cm², capillary l/d=20.

Chapter 5 — Design of Spindles & Spindle Supports (CLO4)

Compliance, optimum bearing span, hollow-spindle sizing, bearing selection, run-out.

Compliance master formula: C = esaddle + eheadstock(1−x/L)² + etailstock(x/L)², where x is measured from the headstock. Nose deflection / run-out: y = δB(1+c/l) + δA(c/l) — front bearing dominates (cantilever amplification 1+c/l). Optimum bearing span λ = l/c ≈ 3–5 (≥2.5).
