MIE 3223 · Machine Tool Technology · ch 3·4·5 · learn the method, not the answer — the values change, the recipe doesn't

How to attack any numerical

Every ch 3/4/5 numerical is the same four-move game. Internalise the moves and a problem you've never seen becomes a problem you've already solved.

The 4 moves — every single time

  1. Convert to cm + kgf first. Before any formula. mm→cm (÷10), kgf·m→kgf·cm (×100), kgf/cm vs kgf/mm (×10). Mixed units is the #1 mark-killer in this paper.
  2. Draw it & label. Sketch the section or free-body, mark every dimension and force you'll use. The picture chooses the formula for you.
  3. Name the type out loud. "This is a closed-section torsion." "This is a carriage-equilibrium slideway." The name pulls the formula — see the chapter tabs.
  4. Substitute, then box with units. Numbers in, one line each, final answer boxed with units. State any "assume suitable data" you used — it's allowed and earns method marks.

Unit conversions you will need

1 kgf·m = 100 kgf·cm (torque/moment) 1 m = 100 cm, 1 mm = 0.1 cm kgf/cm → kgf/mm : ÷10 (stiffness) 1 µm = 10⁻⁴ cm = 10⁻³ mm (deflection, run-out) Esteel = 2.1×10⁶ kgf/cm², ECI = 1.0–1.2×10⁶ kgf/cm² Gsteel = 8×10⁵ kgf/cm² [pav] ≈ ½ [pmax] (bearing pressure)

The mindset that separates a pass from a blank

You are never asked to invent anything. Each problem maps to one of ~14 recipes on the next three tabs. Your job in the exam is diagnosis, not creativity: read the question, spot the trigger words, jump to the recipe, run it. If you've drilled the recipes, the only thing that can go wrong is arithmetic or units — so go slow on those two.

When you're stuck

If you can't see the type: write down what's given and what's asked in symbols. "Given Mt, section dims, G, L → asked τ and θ" instantly says torsion recipe. The given/asked pair almost always names the recipe. If a value is missing, assume the standard one and say so — the prof explicitly allows it.

Universal traps (cost marks on every type)

Mixed units (mm with cm). • Forgetting to halve [pmax] for [pav]. • kgf/cm vs kgf/mm for stiffness. • Reporting radians when degrees asked (×57.296). • Not stating assumptions. • Skipping the free-body sketch and guessing a reaction's direction.

Ch3 · Structures — the 5 recipes

3·A Torsion of a thin section (open vs closed) most-tested

Recognise it

A twisting moment / torque Mt on a beam cross-section (I, channel, box, angle), asks for max shear stress τ and/or angle of twist θ. The picture is a cross-section, not a loaded beam.

Decide first: open or closed?

Open = you could "unroll" it into thin strips (I, channel, L, T). Closed = a continuous tube enclosing area (box, hollow rectangle). This one decision changes everything.

OPEN: It = β·(1/3)·Σ b·t³ β: rect 1.0, channel 1.12, I 1.20 τmax = Mt · tmax / It (tmax = THICKEST element) θ = Mt · L / (G · It) CLOSED: It = 4·A₀² / (∮ ds/δ) A₀ = area at wall MID-LINE τ = Mt / (2·A₀·δ) Mlim = 2·τ·A₀·δ θ = Mt · L / (G · It)

Mindset / first move

Draw the section, split it into rectangles (b × t), and circle the thickest one — that thickness is tmax. Convert mm→cm and Mt to kgf·cm before plugging. Closed beats open ~8–30× in stiffness — if asked "which is better in torsion", it's always closed.

Step recipe (open)

  1. List each thin rectangle as (b, t) in cm. For an I: 2 flanges + 1 web (web height = total − 2×flange thickness).
  2. It = β·(1/3)·Σ b·t³. Don't forget β (1.20 for I).
  3. τmax = Mt·tmax/It, with tmax = thickest element (usually the flange).
  4. θ = Mt·L/(G·It); convert to degrees ×57.296 if asked.

Traps

Forgetting β (gives 46 instead of 55.2). • Using web thickness as tmax instead of the flange. • Web height = full height − 2×flange thickness, not the full height. • For closed sections, A₀ is at the wall mid-line (subtract one wall thickness per outer dimension).

Show worked example ▸

I-section, Mt = 1000 kgf·m, G = 8×10⁵ kgf/cm², L = 100 cm; 140 tall, flange 80×20, web 10 (mm).

What goes where β = 1.20 — it's an I-section (the β table)
rectangles (b, t in cm): two flanges 8 × 2, one web 10 × 1  (web height = 140 − 2×20 = 100 mm = 10 cm)
tmax = 2 cm — thickest element = the flange
Mt = 100 000 kgf·cm — 1000 kgf·m × 100
G = 8×10⁵, L = 100 cm — given

It = β·⅓·Σb·t³ → put β in front, sum (b·t³) over the 3 rectangles:

It = (1.20)·⅓·[ 2·(8·2³) + (10·1³) ] = 1.20·⅓·[128 + 10] = 1.20·46 = 55.2 cm⁴

τmax = Mt·tmax/It → Mt on top, tmax=flange thickness, It from above:

τmax = (100 000)·(2) / (55.2) = 3623 kgf/cm²

θ = Mt·L/(G·It) → Mt and L on top, G·It below:

θ = (100 000)·(100) / (8×10⁵ · 55.2) = 0.2264 rad → ×57.296 = 12.97°

τmax ≈ 3623 kgf/cm², θ ≈ 12.97°. Re-do as a closed box of the same outer size and watch It jump — the exercise that beats the class.

3·B Bending deflection & reduced rigidity (beds, stiffeners) common

Recognise it

A bed/base/table with walls and stiffeners, asks for deflection or reduced bending rigidity EIbr, often comparing perpendicular vs diagonal stiffeners.

Bending stress σ = M·c/I, I = bh³/12 Central deflection δ = P·L³/(48·E·I) Reduced rigidity EIbr = s · E · Iwall (s from Mehta Eq 3.41 perp / 3.42 diag) Stiffness k = F/δ

Mindset / first move

Find the wall's own I first (Iwall = b·h³/12 of one wall), then apply the stiffener factor s. The headline you must remember: diagonal stiffeners ≈ 30× stiffer than perpendicular for the same metal — so if asked "which is better", diagonal wins for both bending and torsion.

Step recipe

  1. Compute the single-wall I = b·h³/12 (cm).
  2. Count stiffeners n from spacing over length.
  3. Apply s₁ (perpendicular, Eq 3.41) or s₂ (diagonal, Eq 3.42) — state the assumed factor.
  4. EIbr = s·E·I; compare the two, report the ratio.

Traps

Using the whole box I instead of the single wall. • Mixing the perpendicular factor with the diagonal one. • Forgetting to convert E to kgf/cm².

Show worked example ▸

Lathe bed, vertical walls 5 cm thick, 100 cm high; E = 1.2×10⁶ kgf/cm² (CI).

What goes where single wall = a tall thin rectangle: b = 100 cm (height), h = 5 cm (thickness, the bending direction)
s = stiffener factor: perpendicular s₁ vs diagonal s₂ (Mehta Eq 3.41 / 3.42)
E = 1.2×10⁶ — cast iron

Iwall = b·h³/12 → b = wall height, h = wall thickness (cube the thin dimension):

Iwall = (100)·(5)³/12 = 100·125/12 = 1042 cm⁴

EIbr = s·E·Iwall → multiply the wall rigidity by the stiffener factor s:

Perpendicular: EIbr ≈ 0.99×10¹⁰  |  Diagonal: EIbr ≈ 3.0×10¹¹ kgf·cm²

Diagonal ≈ 30× stiffer. The number to quote even if you can't recall the exact s factors.

3·C Required stiffness k = F/δ free marks

Recognise it

Gives a cutting force and an allowable deflection, asks for required structural stiffness. (This was Q7(d) on the last paper.)

k = F / δ (watch units: N/mm, or kgf/µm)

Mindset / first move

This is the easiest mark on the paper — it's a one-liner. The only risk is units: keep F and δ consistent.

Traps

δ in mm vs µm. 0.005 mm = 5 µm. Answer 10⁶ N/mm = 1000 N/µm — same number, label it right.

Show worked example ▸

Cutting force 5000 N, allowable deflection 0.005 mm.

What goes where F = 5000 N — the cutting force (numerator)
δ = 0.005 mm — the allowable deflection (denominator) = 5 µm

k = F/δ → force on top, deflection below:

k = (5000) / (0.005) = 1 000 000 N/mm

k = 10⁶ N/mm = 1000 N/µm.

3·D Box column / base deflection common

Recognise it

A hollow box column or base with given outer size + wall thickness, loaded laterally and/or axially, asks for max deflection or angle of slope.

Box section I = (b⁴ − (b−2t)⁴)/12 (square) ; general [BH³−(B−2t)(H−2t)³]/12 Lateral (cantilever) δ = P·h³/(3·E·I) ← usually governs Axial δ = P·h/(E·A) ← usually tiny Slope (moment-area) θ = ∫M dx /(E·I)

Mindset / first move

Get the hollow I first, then compare lateral vs axial deflection — lateral almost always governs, so compute that one carefully and dismiss axial as negligible in one line.

Traps

Subtracting 2t from both dimensions for the hole. • Treating a lateral load as axial. • Aperture/hole → cut local I, deflection rises ~15% (Table 3.4).

Show worked example ▸

CI box column 30×30 cm outer, wall 3 cm, height h = 200 cm, lateral load 200 kgf, E = 1.2×10⁶.

What goes where b = 30 cm (outer), hole side = b − 2t = 30 − 2·3 = 24 cm
P = 200 kgf (lateral), h = 200 cm (column height), E = 1.2×10⁶
use the cantilever formula (lateral load) → it governs over axial

I = (b⁴ − (b−2t)⁴)/12 → outer⁴ minus hole⁴:

I = (30⁴ − 24⁴)/12 = (810 000 − 331 776)/12 = 39 852 cm⁴

δ = P·h³/(3·E·I) → load × height³ on top, 3·E·I below:

δ = (200)·(200)³ / (3 · 1.2×10⁶ · 39 852) ≈ 0.0112 cm ≈ 112 µm

δmax ≈ 112 µm (lateral governs); axial δ = P·h/(E·A) ≈ 1 µm, negligible.

3·E Natural frequency / Perspex model scaling rare

Recognise it

A scale model (Perspex) with a measured frequency, asks for the real structure's frequency — or a frequency-to-avoid range.

fstruct/fmodel = (1/λ)·√(Es·ρ_m / Em·ρ_s) (λ = scale) Frequency to avoid f = n·Z/60, n = 1000V/(πD) (excitation)

Mindset / first move

Put 1/λ outside the root. Look up EPerspex ≈ 0.03×10⁶ kgf/cm². For "frequency to avoid", evaluate f at both extremes of D, V, Z.

Traps

Putting λ inside the root. • Wrong E for Perspex. • Forgetting it's a range (min and max), not one number.

Show worked example ▸

Perspex model, scale 1:10, model frequency 150 Hz → find the steel structure's frequency.

What goes where λ = 10 — the scale (goes outside the root, as 1/λ)
fmodel = 150 Hz — the measured model frequency
Es/Em = 2.1×10⁶ / 0.03×10⁶ = 70 — steel ÷ Perspex
ρm/ρs = 1.2 / 7.8 = 0.154 — Perspex ÷ steel

fs = fmodel·(1/λ)·√(Es·ρm / Em·ρs) → ratios inside the root, 1/λ outside:

fs = (150)·(1/10)·√(70 × 0.154) = 150·0.1·√10.77 = 150·0.1·3.28 = ≈ 49 Hz

fsteel ≈ 49 Hz.

Ch4 · Guideways & Power Screws — the 5 recipes

4·A Slideway reactions & face width most-tested

Recognise it

Cutting-force components Pz, Py, Px + carriage weight G, asks for slideway width or whether the saddle lifts off.

Reactions (two-flat way), b = way spacing, h = tool height, yp = force position: B = Py C = (Pz·yp − Py·h)/b + G/2 A = Pz + G/2 − (Pz·yp − Py·h)/b Face width c' = Areaction / ([pav]·L) Pulling force Q = Px + f·(Pz + Py + G)

Mindset / first move

This is a statics free-body, not a special formula. Draw the carriage, sum forces and moments, get the three reactions, then size width on the largest reaction. Convention: Pz tangential, Py radial, Px axial.

Step recipe

  1. State assumed b, h, yp, f, [pav]=½[pmax] (CI ≈ 25 → 12.5). Say you're assuming them.
  2. Compute B, C, A from equilibrium.
  3. Pick the largest reaction → c' = A/([pav]·L).
  4. Check Q = Px + f(Pz+Py+G). Check no reaction is negative (= lift-off → needs a closed way).

Traps

A negative reaction means lift-off/tipping — don't ignore the sign. • Halve [pmax] for [pav]. • Forgetting to state your assumed geometry (it's expected here).

Show worked example ▸

Turning: Pz=300, Py=130, Px=120 kgf, carriage G=150 kgf, length L=30 cm.

What goes where Pz = 300 (tangential), Py = 130 (radial), Px = 120 (axial), G = 150
assumed: b = 30 (way spacing), h = 20 (tool height), yp = 15 (force position), f = 0.1
[pav] = 12.5 kgf/cm² = ½ of CI [pmax]=25, L = 30 cm

C = (Pz·yp − Py·h)/b + G/2:

C = ((300)(15) − (130)(20))/(30) + 150/2 = (4500−2600)/30 + 75 = 138.3 kgf

A = Pz + G/2 − (Pz·yp − Py·h)/b → the biggest reaction, sizes the width:

A = 300 + 75 − (1900/30) = 375 − 63.3 = 311.7 kgf (largest)

c' = A/([pav]·L) and Q = Px + f(Pz+Py+G):

c' = (311.7)/((12.5)(30)) = 0.83 cm  |  Q = 120 + 0.1(300+130+150) = 120 + 58 = 178 kgf

width ≈ 0.83 cm (round to 1 cm); Q ≈ 178 kgf. All reactions positive → no lift-off.

4·B Pulling force: conventional vs ball common

Recognise it

Same cutting components, asks to compare the pull for a sliding slideway vs a recirculating-ball way.

Conventional Q = Px + f·(Pz + Py + G) (sliding f ≈ 0.1–0.15) Ball way Q = Px + n·Fc + (fr/req)·N (rolling fr ≈ 0.001 cm) req = r/k (k≈1.4), N = Pz + G

Mindset / first move

The whole point: rolling friction is tiny, so the ball way's pull collapses to almost just Px. Compute both, state the % saving. Px is irreducible — the saving is all in the friction term.

Traps

Using sliding f for the ball way. • Forgetting the small constant n·Fc term. • Mixing cm/mm for the ball radius.

Show worked example ▸

Pz=300, Py=130, Px=120 kgf, carriage G=150 kgf; ball dia 10 mm.

What goes where Px = 120 (axial — irreducible, in both), sum (Pz+Py+G) = 580
conventional f = 0.15 (sliding) · ball: fr = 0.001 cm (rolling), req = 0.357, n = 4 faces, Fc = 0.5
N = Pz+G = 450 (normal load on the balls)

Conventional: Q = Px + f(Pz+Py+G):

Q = 120 + (0.15)(580) = 120 + 87 = 207 kgf

Ball: Q = Px + n·Fc + (fr/req)·N → rolling term is tiny:

Q = 120 + (4)(0.5) + (0.001/0.357)(450) = 120 + 2.0 + 1.26 = 123 kgf

207 vs 123 kgf → ~40% lower. The saving is all in the friction term; Px can't be reduced.

4·C Lead screw: pressure, shear, pitch error common

Recognise it

An Acme lead screw with OD, pitch, effective dia, nut length + cutting components, asks for average pressure, max shear stress, pitch error.

Axial pull Q = Px + f·(Pz + Py + G) pav = 2Q/(π·d·L) (d = effective dia, L = nut length) σ = 4Q/(π·dm²), τ = 16Mt/(π·dm³) (dm = minor = OD − pitch) τmax = ½·√(σ² + 4τ²) Pitch error Δt = Q·t/(E·Am), Am = π·dm²/4

Mindset / first move

Order matters: Q first, then pav, then torque (needs helix angle λ and α=14.5° for Acme), then stresses on the minor diameter, then pitch error last. Each result is usually tiny — the screw is accuracy-governed, not strength-governed.

Traps

Using effective dia for stresses (use minor dm = OD − pitch). • Forgetting α=14.5° for Acme. • pav uses effective dia and nut length.

Show worked example ▸

Acme lead screw: OD=70, effective d=65, pitch t=10, nut L=100 (mm); Pz=150, Py=50, Px=45 kgf, f=0.2, G=50.

What goes where Q built from: Px = 45, f = 0.2, sum (Pz+Py+G) = 250
pav uses: d = 6.5 cm (effective dia), L = 10 cm (nut length)
stresses use: dm = OD − pitch = 7−1 = 6.0 cm (minor dia), not the effective dia!
pitch error uses: t = 1 cm, E = 2.1×10⁶, Am = πdm²/4 = 28.27 cm²

Q = Px + f(Pz+Py+G):

Q = 45 + (0.2)(250) = 95 kgf

pav = 2Q/(π·d·L) → effective dia & nut length:

pav = 2(95)/(π·6.5·10) = 0.93 kgf/cm²

τmax = ½√(σ²+4τ²) with σ, τ on the minor dia dm=6.0; Δt = Q·t/(E·Am):

τmax ≈ 2.52 kgf/cm²  |  Δt = (95)(1)/((2.1×10⁶)(28.27)) = 1.6×10⁻⁶ cm = 0.016 µm

pav=0.93, τmax=2.52 kgf/cm², Δt≈0.016 µm — all safe (screw is accuracy-governed).

4·D Hydrostatic pad: pressure, capacity, restrictor harder

Recognise it

A hydrostatic pad (pad area, pocket area, pump pressure, capillary l/d), asks for pocket pressure, load capacity or restrictor size.

Load P = p₀·A·CL CL = load coefficient ≈ 0.3–0.55 p₀ = P/(A·CL) (start here — always unambiguous) Optimum stiffness at p₀/pp = 2/3 Capillary crc = (CF/h³)·(pp−p₀)/p₀, lc = 20·dc

Mindset / first move

Always begin with CL and p₀ = P/(A·CL) — those are the marks you can always get. The restrictor algebra is bonus; if time is short, write the formula chain and the p₀/pp=2/3 result and move on.

Traps

Over-investing time here — it's low-frequency, high-effort. • Mehta's flow coefficient carries an embedded unit constant; show the chain, don't chase a perfect number.

Show worked example ▸

Hydrostatic pad 5×8 cm, pocket 1×6 cm, load P=50 kgf, pump pp=40 kgf/cm², capillary l/d=20.

What goes where A = 5×8 = 40 cm² (pad area), CL = 0.542 (load coeff from pad/pocket sizes)
P = 50 kgf (the load to support) → goes on top to find p₀
pp = 40 (pump pressure) → restrictor must drop pp − p₀

p₀ = P/(A·CL) → load ÷ (area × load-coeff). Always start here:

p₀ = (50)/((40)(0.542)) = 50/21.7 = 2.31 kgf/cm²

Restrictor drop = pp − p₀ = 40 − 2.31 = 37.7 kgf/cm²; size via crc = (CF/h³)(pp−p₀)/p₀ with lc=20·dc.

p₀=2.31 kgf/cm², CL=0.542 — the graded quantities. (Optimum stiffness if p₀/pp=2/3.)

4·E Recirculating ball screw load rare

Recognise it

A ball screw (ball dia, total balls, rpm, service life), asks for load per ball and static/dynamic pull.

Per ball P = 200·db² (db in cm) Static Q = 100·Zr·db², Zr = 0.7·Zt Dynamic Qdyn = Q/k, k = 0.9·(60·T·n·f/10⁷)^(1/3), f = Zt/2

Mindset / first move

Three formulas in a chain. Remember Zr = 0.7 Zt (not all balls are loaded) and watch the cube root in k.

Traps

Using full Zt instead of 0.7 Zt. • Cube root forgotten. • db in mm not cm.

Show worked example ▸

Ball screw: ball dia db=10 mm, total balls Zt=120, n=40 rpm, service life T=10000 h.

What goes where db = 1 cm — 10 mm → cm (must be cm, gets squared)
Zt = 120 total → loaded Zr = 0.7×120 = 84 (only 70% carry load)
f = Zt/2 = 60, n = 40, T = 10000 → feed the cube root in k

Per ball: P = 200·db²:

P = 200·(1)² = 200 kgf

Static: Q = 100·Zr·db² (use Zr=0.7Zt, not Zt):

Q = 100·(84)·(1)² = 8400 kgf

Dynamic: k = 0.9(60·T·n·f/10⁷)^⅓, Qdyn = Q/k:

k = 0.9·(60·10000·40·60/10⁷)^⅓ = 0.9·(144)^⅓ = 0.9·5.24 = 4.72 → Qdyn = 8400/4.72 = 1781 kgf

per ball 200, static 8400, dynamic ≈1781 kgf.

Ch5 · Spindles — the 4 recipes

5·A Machine-tool compliance most-tested

Recognise it

Work between centres, radial force at a position, given stiffnesses of saddle / headstock / tailstock, asks for compliance or cutting-point displacement.

C = 1/Ks + (1/KA)(1 − x/L)² + (1/KB)(x/L)² (x measured FROM the headstock A; KA headstock, KB tailstock) Displacement y = C · Py

Mindset / first move

First fix x/L — and watch the wording: "200 mm from the tailstock" means x = L − 200. Compliances add in series. Then y = C·Py. Only the radial Py matters for diameter error; Pz is a distractor.

Step recipe

  1. Convert all stiffnesses to the same unit (kgf/cm→kgf/mm ÷10).
  2. Find x from the headstock; compute x/L.
  3. C = sum of the three terms.
  4. y = C·Py. Box it in µm.

Traps

Measuring x from the wrong end. • Using Pz instead of Py. • Mixing kgf/cm with kgf/mm.

Show worked example ▸

Between centres, L=750 mm; tool 200 mm from tailstock; Ks=3000, KA=4000, KB=2500 kgf/mm; Py=150 kgf.

What goes where x = L − 200 = 550 mm — "from tailstock" → measure from headstock! x/L = 0.733
Ks = 3000 (saddle), KA = 4000 (headstock), KB = 2500 (tailstock) — all kgf/mm
(1−x/L) = 0.267 pairs with KA; (x/L) = 0.733 pairs with KB
Py = 150 — radial only (Pz would be a distractor)

C = 1/Ks + (1/KA)(1−x/L)² + (1/KB)(x/L)²:

C = 1/3000 + (1/4000)(0.267)² + (1/2500)(0.733)² = 3.33×10⁻⁴ + 0.18×10⁻⁴ + 2.15×10⁻⁴ = 0.566 µm/kgf

y = C·Py:

y = (0.566)(150) = 85 µm

C ≈ 0.566 µm/kgf; displacement ≈ 85 µm.

5·B Spindle nose deflection & optimum span common

Recognise it

Spindle on two bearings + overhang, asks for nose deflection or optimum bearing span λ.

Nose deflection y = δB(1 + c/l) + δA(c/l) (front bearing B dominates) δ = R/K at each bearing; reactions from beam statics Optimum span λ = l/c ≈ 3–5 (must be ≥ 2.5 for anti-friction)

Mindset / first move

The front bearing dominates (the 1+c/l factor amplifies it). Get bearing reactions by statics, δ=R/K, then combine. For "optimum span" just quote λ≈3–5 and the reason: bending rises with span, support-compliance falls, so the sum has a minimum.

Traps

Swapping front/rear bearing. • Forgetting the 1+c/l amplification. • Reporting λ<2.5 (not allowed for ball bearings).

Show worked example ▸

Spindle: front-bearing yield δB=42.2 µm, rear δA=9.17 µm, overhang c=1, span l=3.

What goes where δB = 42.2 µm — FRONT bearing (gets the 1+c/l boost — it dominates)
δA = 9.17 µm — rear bearing (only the c/l term)
c/l = 1/3 = 0.333 → so (1+c/l) = 1.333

y = δB(1+c/l) + δA(c/l) → front term amplified by 1+c/l:

y = (42.2)(1.333) + (9.17)(0.333) = 56.3 + 3.1 = ≈ 59 µm

y ≈ 59 µm; λopt = l/c ≈ 3 (bending rises, support-compliance falls → sum minimises).

5·C Run-out from bearing clearance common

Recognise it

Two bearings with radial clearances + nose overhang, asks for radial run-out of the nose.

Run-out = δB(1 + c/l) + δA(c/l) (same form as nose deflection) δB = front clearance, δA = rear; worst case = opposite directions

Mindset / first move

Identical formula to 5·B but the δ's are clearances, not load deflections. Take them in opposite senses for the worst case. Front clearance dominates.

Traps

Adding clearances same-sign (under-estimates). • Using rear as front.

Show worked example ▸

Bearings l=20 cm apart; clearances 6 µm (front) & 4 µm (rear); nose overhang c=8 cm.

What goes where δB = 6 µm — FRONT clearance (amplified by 1+c/l)
δA = 4 µm — rear clearance (only c/l)
c/l = 8/20 = 0.4 → (1+c/l) = 1.4; take clearances opposite for the worst case

run-out = δB(1+c/l) + δA(c/l) (same form as nose deflection):

= (6)(1.4) + (4)(0.4) = 8.4 + 1.6 = 10 µm

max radial run-out = 10 µm.

5·D Journal bearing: p, v, pv, film harder

Recognise it

A sleeve / journal bearing (dia, load, rpm, l/d), asks to select material or check p, v, pv / film thickness.

Pressure p = P/(l·d) Speed v = π·d·n/60 pv product (check against [pv]) Clearance c = ψ·d/2, film hmin = c(1 − ε)

Mindset / first move

Compute p, then v, then pv — those three route you to the material table. High v + low p → tin bronze / white metal. Don't attempt the full hydrodynamic design unless explicitly asked.

Traps

Using πdl (cylinder area) for projected area — it's l·d. • For heat balance, free surface A = 20·d·l, not πdl. • Over-investing in the rare full-design version.

Show worked example ▸

Journal bearing: d=80 mm, load P=30 kgf, n=2500 rpm, l/d=1.

What goes where d = 8 cm (for pressure), = 0.08 m (for surface speed); l = d = 8 cm (l/d=1)
P = 30 kgf → projected area = l·d = 8×8 = 64 cm² (NOT πdl)
n = 2500 rpm → goes in v = πdn/60

p = P/(l·d) → load ÷ projected area:

p = (30)/((8)(8)) = 0.47 kgf/cm²

v = π·d·n/60 → d in metres:

v = π·(0.08)·(2500)/60 = 10.5 m/s → pv = 0.47×10.5 ≈ 4.9

pv ≈ 4.9; high v, low p → tin bronze; within [p], [pv].
